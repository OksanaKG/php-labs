# ⚡ Швидкий старт - Кінотеатральна система v3

## 🎯 Що було додано?

### 📊 Основні компоненти:
1. **Галерея фільмів** - красиві картки з постерами
2. **Коментарії** - користувачі залишають думки про фільми (1-10 зірок)
3. **Реакції** - like/dislike для фільмів
4. **Голосування** - опитування на фільмах
5. **Активності** - голосування за благодійні фільми та спецсеанси
6. **Квитки** - повна система покупки квитків з вибором місць у залі
7. **Сеанси** - можливість створювати покази для фільмів
8. **Статистика** - лічильники коментарів, лайків, голосів

## 🧪 Тестування функцій

### 1️⃣ Коментарі
1. Увійти як користувач
2. Перейти на галерею `/index.php?route=movie/gallery`
3. Клікнути на фільм
4. Внизу залишити коментар з рейтингом (1-10)
5. Коментар з'явиться в списку
6. Можна редагувати свій коментар

### 2️⃣ Реакції (Like/Dislike)
1. На сторінці деталей фільму натиснути "👍 Мне нравится"
2. Кількість лайків змінюватиметься (AJAX без перезавантаження)
3. Можна змінити реакцію на дизлайк

### 3️⃣ Голосування (Polls)
1. На сторінці деталей фільму вибрати варіант у опитуванні
2. Голос враховується автоматично
3. Видна кількість голосів для кожного варіанта
4. Показується прогрес-бар

### 4️⃣ Активності
1. Перейти на `/index.php?route=activity/list`
2. Вибрати активність (благодійна, дитяча, спеціальна)
3. Клікнути "Голосувати" 
4. Лічильник голосів оновиться
5. Можна скасувати голос (toggle)

### 5️⃣ Покупка квитків
1. На сторінці деталей фільму перейти на сеанс
2. Клікнути "Купити квитки"
3. Вибрати місця з сітки (зелене = вільне, сіре = забруднене)
4. Сума розраховується автоматично
5. При кліку "Завершити покупку" квитки генеруються

### 6️⃣ Створення сеансів (адмін)
1. Увійти як адміністратор
2. На сторінці деталей фільму (для адміна буде кнопка)
3. Або посилання: `/index.php?route=movie/create_screening?movie_id=1`
4. Заповнити дату/час, залу та ціну
5. Сеанс з'явиться на сторінці деталей

```

      ↓

      ↓
```

## ⚠️ Важні примітки

1. **База даних** видаляється при першому запуску і створюється заново
2. **Адміністратор** визначається за логіном `admin` або полем `is_admin`
3. **Квитки** генеруються з форматом `TKT-YYYYMMDDHHmmss-XXXX`
4. **Коментарі** заменяються (один коментар на користувача на фільм)
5. **Голоси** тоже унікальні (один голос на користувача на опитування)
