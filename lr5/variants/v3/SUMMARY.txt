                            КАК ЗАПУСТИТИ
================================================================================

準ВІ-ВИМОГИ:
  • PHP >= 7.4
  • SQLite embedded в PHP (зазвичай вже є)
  • Браузер з підтримкою HTML5

ЗАПУСК:
  1. cd d:\Бекенд\php-labs\lr5\variants\v3
  2. php -S localhost:8080
  3. Відкрити http://localhost:8080 в браузері

ПЕРШИЙ ЗАПУСК:
  • database/app.db автоматично створюється
  • schema.sql виконується (12 нових таблиць)
  • Seed data вставляються (10 фільмів, 3 активності)

ТЕСТУВАННЯ:
  1. Зареєструйтеся як новий користувач
  2. Натисніть "gallery" для перегляду карток
  3. Клікніть на фільм для деталей
  4. Залиште коментар
  5. Додайте реакцію
  6. Голосуйте в опитуванні
  7. Купіть квитки
